Offiziele Homepage von John Klein

Hallo! Dies ist meine offizielle Homepage auf welcher ich dir zeige was ich so mache. Wenn du von einem Unternehmen bist, kann ich dir den "Sprachen" Tab empfehlen. Diesen kannst du in der Hotbar finden oder du klickst hier.

Was siehst du oben?

Lass mich dir erklären was du oben siehst. Auf dem Bild ist das HTML Dokument des Tabs "Links" zu sehen. Als erstes habe ich den Tab mit "Links" betitelt und mein CSS stylesheet verlinkt. Dies habe ich in dem Kopf der Webseite eingebunden. Danach habe ich 3 Einteilungen erstellt:

  1. ID top

    2. Diese Teilung ist so gesetzt dass sie auf der Webseite oben platziert ist und hat den Text Zentriert.

  2. div class="elpic"

    3. Hier habe ich die Klasse elpic zu der Teilung zugewiesen um einen Hintergrund einzufügen. Ich könnte auch eine ID zuweisen, was ich aber für künftige Arbeitsschritte nicht tat.

  3. div class="name"

    4. In diesem Schritt habe ich dem Namen eine Klasse in meiner CSS stylesheet zugewiesen, damit er mittig im Bild auf der Seite zu sehen ist.

  4. Navigation bar aka Hotbar

    5. Dann wird es interessant: Die Navigation oder auch Hotbar. Hier habe ich eine geordnete Liste erstellt und habe die Links der anderen Tabs eingefügt um durch die Webseite mit einem GUI (Grafical User Interface) zu navigieren. Die Hotbar Schaltflächen haben verschiedene aussehen, welche ich mit den IDs bestimmt habe. Die Klasse "active" ist jeweils dem aktiven Tab zugewiesen.

  5. Links und Beschreibungen

    6. Hier wird es einfacher und mehr copy paste Arbeit. Zuerst habe ich die ID social dem Element zugewiesen welches die Schaltflächen und Beschreibungen enthält. Dann habe ich Schaltflächen eingebunden welche ich in der CSS stylesheet welche oben verlinkt sind und habe Beschereibungen darunter eingefügt. Diesen Schaltflächen habe ich Links zugewiesen welche in den "p" (paragraph) Tags beschrieben sind.

    In dem Bild ist ein kleiner Fehler eingebaut, wenn er dir aufgefallen ist dann lass es mich gerne wissen.

Ich hoffe dir gefällt diese Seite. Wenn du Feedback hast kannst du mich gerne unter dem Kontakte Tab kontaktieren.
